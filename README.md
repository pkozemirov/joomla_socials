# joomla_socials
Авторизация через соцсети для joomla 2.5+ (может и ниже, не проверялось)

Поддерживаемые соцсети:
* Facebook
* Instagram
* Mail Ru
* Microsoft (live)
* Odnoklassniki
* Vkontakte
* Yandex

В ближайшем будущем планируется более детальный разбор OAuth-авторизации и соответствующий модуль на NodeJS.
